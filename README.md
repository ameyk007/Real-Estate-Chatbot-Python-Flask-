Overview
The Real Estate Chatbot is a web-based application developed using Python and Flask. It helps users get quick information related to buying or renting properties, preferred locations, budget ranges, and contacting real estate agents. The chatbot provides instant responses, improving customer engagement and reducing manual workload for real estate businesses.

Objectives
Automate customer interaction for a real estate company
Provide instant replies to common property-related queries
Store user inquiries in a database for future reference

Technologies Used
Python
Flask Framework
HTML, CSS (Frontend)
SQLite / MySQL (Database)
JavaScript (Optional for UI interaction)

Features
User-friendly chat interface
Quick responses to property-related questions
Supports queries for buying and renting properties
Stores customer inquiries securely in a database
Easy integration with existing real estate platforms

Application Use Case
This chatbot can be used by real estate companies to:
Handle customer queries 24/7
Reduce response time
Improve customer satisfaction

Maintain a record of customer requirements
