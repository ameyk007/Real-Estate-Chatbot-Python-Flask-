# Real-Estate-Chatbot-Python-Flask-
Overview
The Real Estate Chatbot is a web-based application developed using Python and Flask. It helps users get quick information about buying or renting properties, locations, budget ranges, and contacting agents.

Objectives
Automate customer interaction for a real estate company
Provide instant replies to common property-related queries
Store user inquiries in a database for future reference
